# /review

**ultrathink**

指定されたコードに対して、品質・セキュリティレビューを実行してください。
問題点が検出された場合は、修正案を提示し、修正を行います。すべての問題がクリアされるまで繰り返します。

## 1. 品質・セキュリティレビューを実行する

- 必ず `quality-reviewer` エージェントを使用する
- 静的解析、テスト実行、セキュリティ検査を網羅的に実施する
- 問題の重要度（重大/警告/情報）を分類する

## 2. 検出された問題を修正する

- 重大な問題から優先的に修正する
- Python コードの場合は `python-expert` エージェントを使用する
- Terraform コードの場合は `terraform-expert` エージェントを使用する
- 修正後、再度 1 のレビューを実行する

## 3. 最終確認

- すべての静的解析エラーが 0 件であることを確認する
- すべてのテストが通過していることを確認する
- セキュリティ警告がないことを確認する
