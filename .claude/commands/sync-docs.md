# /sync-docs

**ultrathink**

コードの実装内容とドキュメントを同期してください。

## 手順

1. 対象コードの現在の実装を分析する
2. 関連するドキュメント（README, CLAUDE.md, docstring など）を確認する
3. 実装とドキュメントの差分を特定する
4. ドキュメントを実装に合わせて更新する

## 同期対象

- README.md のセクション
- CLAUDE.md の設定・規約
- コード内の docstring
- API ドキュメント

## ルール

- 実装が正であり、ドキュメントを実装に合わせる
- 削除された機能はドキュメントからも削除する
- 新機能はドキュメントに追記する
- 変更履歴は記載しない（「〇〇を追加」などは書かない）

## 対象

$ARGUMENTS

（指定がない場合はプロジェクト全体）
