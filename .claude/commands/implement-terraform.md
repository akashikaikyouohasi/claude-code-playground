# /implement-terraform

**ultrathink**

指定されたタスクに対して、以下の順で実装からレビューまで完結させてください。
問題点があれば修正を行い、すべてクリアするまで繰り返します。

## 1. インフラ実装

- 必ず `terraform-expert` エージェントを使用する
- プロジェクト内容を詳細に理解したうえで、要件に基づいて実装する
- モジュール構造に従い、セキュリティベストプラクティスを適用する
- 静的解析・セキュリティチェックをすべて通過させる

## 2. 品質レビュー

- 必ず `quality-reviewer` エージェントを使用する
- 実装要件に抜け漏れがないか徹底的にレビューする
- セキュリティリスクなど潜在的な問題がないか確認する
- 問題が検出された場合は、1 に戻って修正を行う
